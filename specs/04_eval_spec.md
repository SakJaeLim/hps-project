# 04. 평가 명세

1차 목표는 **기술 통합·작동**(ADR-0001). 성능 경신이 아니므로 재취급률은 참고 비교지표로 둔다.

| 지표 | 방법 | 목표/기준 |
|---|---|---|
| RL 모델 통합 작동 | 유효 적재계획 생성률 · 전 흐름 동작 | 핵심 — 정상 작동 |
| 제약 위반율 | 검증 에이전트 Cypher | < 1% |
| 권고 응답속도 | 평균 응답시간 | ≤ 5초 |
| 재취급 (참고) | RL/Greedy vs 랜덤·단순룰 (시뮬) | 기준선 대비 비교 |
| SLM 도메인 품질 | 파인튜닝 전/후 정성 비교 | 환각↓·근거↑ |

평가 시나리오 3종(도메인 전문가 정의): 일반/피크/특수화물(DG·Reefer).

## 향후 실험: RL vs CP 엔진 벤치마크 (ADR-0001)
동일 시뮬 인스턴스 세트에서 RLStrategy vs CPStrategy 비교(기준선 GreedyStrategy 포함).
- 지표: 재취급 수 · 제약 위반율 · 계획 산출 시간 · (가능 시) 최적성 갭
- 프로토콜: 고정 시드 · 홀드아웃 인스턴스 · 다회 반복(평균±분산)
- 판정: **CP가 유의미하게 우월할 때에만** 기본 엔진 교체. 아니면 RL 유지(현 엔진).
